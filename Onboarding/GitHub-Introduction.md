Welcome to this quick video introduction to GitHub.I'm going to walk you through creating a fork, making a change, commiting those changes and finally creating a pull request to include those changes.
The change you will be making is adding your contact card. The properties aren't all required but it would be helpful to include as much as you are comfortable with to make communication as smooth as possible.
To get started, create a professional GitHub account or log in to an existing one.
Navigate to the home page of the main repository for Solar Knights Racing by searching for "SKR-Core" at the top.
Make sure to star this repository for later.
To propose changes without write permissions you must create a fork.
To create a fork click the `fork` button at the top right.
Proceed with default options and click `Create fork`
Navigate to Operations/Administration/Contacts
View someone's file to see how the contact card is supposed to look
Then go to README.md and click `copy raw contents`
Create a new file called 'your first name' dash 'your last name' dot md
   e.g. Smith-John.md
fill out your properties
commit your changes by clicking `commit changes` at the top right
To include your changes in the master branch of the original repository, you must open a pull request.
Navigate to the home page of your forked repository. click `Contribute` and then `Create Pull Request`
You can write an short introduction including things like your major, what you're interested in, or your goals if you would like. Then, click `create pull request`
